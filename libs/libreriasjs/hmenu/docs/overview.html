DynarchMenu: high quality menus

This is a versatile DHTML menu script, providing keyboard shortcuts, CSS-customizable look, cross-browser and cross-platform support, cross frame menus, icon support, tooltips support, smooth menu shadows in all supported browsers (except IE 5.0), friendly to search engines, many high quality color themes, etc. Despite all these features, the JS code is small and extremely easy to setup and customize: it requires absolutely no JS knowledge whatsoever.

New in 2.9

» Support for scrolling improves usability of long menus. Enable it by passing “scrolling: true” to the setup function.

New in 2.8

» Support for cross frame menus. See the samples: horizontal layout and vertical layout.
» Fixes for the iframe approach (see the sample).

Other features are: check-boxes and radio buttons (see the “Preferences” submenu), but given their dynamic nature one needs to code a few JavaScript lines for that. Our documentation includes such examples. Support for PNG icons in IE 5.5+ is possible with a very small and free additional script (PieNG) (the script is also included in the DynarchMenu distribution, for your convenience).

The full features list is rather long so we decided to put together a separate section about this. Click here or use the menu to go to it.

Supported browsers

DynarchMenu is currently tested in Internet Explorer for Windows 5.0, 5.5. 6.0, Mozilla, FireFox, Netscape 7 for Windows and Linux, Opera 7 for Windows and Linux and Konqueror 3.2 for Linux (Apple Safari is also known to work). The menu is also assumed to work with other Gecko-based browsers such as Camino, Kameleon, Galeon, Beonex, etc.
