The above menu defines 2 custom timeouts. The “electric” parameter is set to 1000, which means that all menus will close if the mouse was outside for 1 second (1000 milliseconds), and the “timeout” parameter is set to 0 which means that submenus will open instantly when the mouse hovers an item that has a submenu.

The default values are:

  • electric = false (which means that the menu won't show “electric” behavior, so items in the toplevel menu bar require a click in order to display submenus.
  • timeout = 150 milliseconds, which we figured out to be a sane value for opening submenus on “mouseover”.
