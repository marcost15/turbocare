1. How to Create a vertical bar chart

1.1 Create a bar chart object

We first include the chart library in our project, and then create a chart object of dimensions 500 x 250 pixels. If not specified, the image dimensions will be defaulted to sensible values. There are currently 3 types of charts : VerticalChart, HorizontalChart (both of which are bar charts) and PieChart.

	include "libchart/libchart.php";

	$chart = new VerticalChart(500, 250);

1.2 Create some bars

We add 4 bars to our chart. Each bar is a (label, value) couple, that we call here a Point.

	$chart->addPoint(new Point("Jan 2005", 273));
	$chart->addPoint(new Point("Feb 2005", 321));
	$chart->addPoint(new Point("March 2005", 442));
	$chart->addPoint(new Point("April 2005", 711));

1.3 Display the chart

We set the title and then render the chart to a PNG image.

	$chart->setTitle("Monthly usage for www.example.com");
	$chart->render("generated/demo1.png");

...et voilà!

2. How to Create a horizontal bar chart

Creating a horizontal bar chart is similar to the previous example.
Here is the source code and the corresponding result.

	$chart = new HorizontalChart(500, 170);

	$chart->addPoint(new Point("/wiki/Instant_messenger", 50));
	$chart->addPoint(new Point("/wiki/Web_Browser", 83));
	$chart->addPoint(new Point("/wiki/World_Wide_Web", 142));

	$chart->setTitle("Most visited pages for www.example.com");
	$chart->render("generated/demo2.png");

3. How to Create a pie chart

In order to create a pie chart, we enter 3 sampling points and their associated absolute values. Percentages are computed automatically.

	$chart = new PieChart(500, 250);

	$chart->addPoint(new Point("Mozilla Firefox (80)", 80));
	$chart->addPoint(new Point("Konqueror (75)", 75));
	$chart->addPoint(new Point("Other (50)", 50));

	$chart->setTitle("User agents for www.example.com");
	$chart->render("generated/demo3.png");
